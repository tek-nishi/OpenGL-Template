# OpenGL Template for C++

## 概要
OpenGL 1.1あたりを始めようとする時にいい感じの雛形が見当たらないので作りました。

## 対応OS 
+ Windows

  VisualStudio Community 2017で構築していますが、ビルドオプションを調整すれば古いバージョンでもビルドできるでしょう

+ macOS

  最新版のXcodeをお使いください

## 利用した外部ライブラリ
+ OpenGL 1.1
+ GLFW 3.2.1
+ GLEW
